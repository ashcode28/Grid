#🎨Grid Pattern

##📌Description
Simple grid pattern using HTML & CSS Grid with colored boxes.

## 🚀 Features
* 6-column grid (`repeat(6, 70px)`)
* Square boxes (70×70)
* 3 colors: orange, peachpuff, brown
* Repeating pattern

## 🛠️ Tech
HTML, CSS (Grid)

## ▶️ Run
Open the `.html` file in a browser.

## 💡 Concepts
CSS Grid, layout, class styling
